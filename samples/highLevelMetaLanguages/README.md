High-level meta-languages
=========================

This sample project illustrates how to create your own language-definition languages (aka meta-languages) in MPS. This way you can provide simplified or somehow tuned ways to define the structure and the editors of a language.

The "simpleStructure" language defines a fairly straightforward language to define structure of a language, while the "simpleEditor" editor defines a presentation-definition language. The generators of both of these languages codify how to translate simple structure and editor definitions into proper ones and the "sampleLanguage" serves as an example of a language defined by means of the simplified languages.
